# Typescript learning

# Build

**Build app.ts into javascript**

```bash
# Using tsc
tsc app.ts

# Using yarn
yarn tsc <file>
```

# Testing

**Load the live server**
In a separate terminal type:

```bash
yarn start
```

# Notes

Enter watch mode (auto compile)

```bash
yarn tsc app.ts --watch
```
